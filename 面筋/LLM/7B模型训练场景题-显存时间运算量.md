## 核心面筋要点

### 🔸 面试趋势：场景题取代八股文

- 面试官不问原理了，直接抛真实业务场景
- 光背八股文完全覆盖不了，必须真正动手干过

### 🔸 高频场景题举例

| 问题 | 考点 |
|------|------|
| 怎么选基座模型？ | 模型选型决策能力 |
| 数据怎么配比？ | 数据工程经验 |
| 训练崩了怎么救？ | 故障排查与调参能力 |
| SFT数据需要多少条？ | **不是要数字，而是考数据质量和多样性的判断标准** |

### 🔸 技术踩坑重灾区

- 数据没清洗到位 → 训练完全白干
- Loss震荡不知道怎么调 → 卡住几周
- 效果不升反降找不出原因 → 项目延期
